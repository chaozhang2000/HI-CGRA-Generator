# CGRA-Generator

## Compile chisel code

* Install `mill`. Refer to [the Manual section in this guide][mill]. mv the mill to ~/
* Run `make` to generate verilog code. The output file is `build/TopMain.v`.

[mill]: https://com-lihaoyi.github.io/mill/ 

## Quick start

```
cd CGRA-Generator
sudo chmod 777 ./mill
make
```
